# TrueNAS Storage Lab

## Overview

A home lab project created to practice network storage, file sharing, and server storage management.

## Environment

- TrueNAS
- NAS Server
- Network Storage
- Storage Drives

## Completed Tasks

### Storage Configuration
- Installed and configured TrueNAS
- Created storage pools
- Managed disks and storage resources
- Practiced storage organization

### File Services
- Configured network file sharing
- Tested client access
- Managed permissions

### Server Administration
- Monitored storage health
- Practiced basic server maintenance
- Learned NAS concepts

## Skills Practiced

- Storage Administration
- NAS Management
- File Sharing
- Permissions
- Server Basics

## Tools Used

- TrueNAS
- Network Storage
- Server Hardware
